# Interactive Quiz App

An engaging and dynamic quiz application built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.  
The app includes multiple quiz categories, countdown timer, score tracking, and progress indicators for an interactive user experience.

---

## 🚀 Features

- Multiple quiz categories
- Real-time score tracking
- Countdown timer for each question
- Progress bar indicator
- Responsive design
- Interactive UI with animations
- Restart quiz functionality

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON
